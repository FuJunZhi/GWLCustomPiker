# GWLCustomPiker
自定义外观的pikcerView，使用方法和UIPikcerView基本一致
## Requirements

* iOS 7.0+
* Xcode 6+

## Installation

### CocoaPods 
pod 'GWLCustomPiker'

## Usage
import 'GWLCustomPiker.h'

```Objective-C
GWLCustomPikerView *customPikerView  = [[GWLCustomPikerView alloc]init];
customPikerView.frame = ?;
customPikerView.dataSource = ?;
customPikerView.delegate = ?; 
```
